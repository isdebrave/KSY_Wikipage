## 📌 실행 방법

1. 깃헙 클론
   ```
   git clone https://github.com/isdebrave/KSY_Wikipage.git
   ```
2. 필요한 패키지 설치
   ```
   npm i
   ```
3. 실행
   ```
   npm start
   ```
